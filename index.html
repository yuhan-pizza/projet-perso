Jump Frog : Fait par Mohamed Makhlouf

À propos du projet

Jump Frog est un jeu de plateforme dynamique où vous contrôlez une grenouille qui doit naviguer à travers des obstacles pour atteindre le drapeau de fin de niveau.

Le jeu propose des mécaniques variées avec une difficulté progressive pour un challenge toujours plus intense.

Comment ça marche

Contrôles

Utilisez les touches A/D ou ←/→ pour déplacer la grenouille et Espace pour sauter. Attention aux obstacles !

Objectif

Atteignez le drapeau à la fin de chaque niveau. La difficulté augmente progressivement avec de nouveaux obstacles et plateformes mobiles.

Score

Collectez des bananes pendant votre parcours pour augmenter votre score. Chaque banane rapporte des points bonus !

Extrait de code

Voici un extrait du code JavaScript qui gère les contrôles et les collisions :

// Gestion des contrôles
document.addEventListener('keydown', (e) => {
    if (e.key === 'ArrowLeft' || e.key === 'a') {
        frogDirection = -1; // Déplacement gauche
    }
    if (e.key === 'ArrowRight' || e.key === 'd') {
        frogDirection = 1; // Déplacement droite
    }
    if (e.code === 'Space' && isGrounded) {
        frogVelocity = jumpForce; // Saut
    }
});

// Détection de collision avec les bananes
function checkBananaCollision() {
    bananas.forEach((banana, index) => {
        if (Math.abs(frogX - banana.x) < 20 && 
            Math.abs(frogY - banana.y) < 20) {
            score += 10;
            bananas.splice(index, 1);
            playCollectSound();
        }
    });
}

// Vérification de la victoire (drapeau)
function checkWinCondition() {
    if (Math.abs(frogX - flagX) < 30 && 
        Math.abs(frogY - flagY) < 50) {
        levelComplete();
    }
}
